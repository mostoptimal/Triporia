# Triporia

Essential apps & websites for travelers. Pre-trip planning and on-the-ground apps for 60+ countries with tips, live clocks, and a travel checklist.

## Features

- **60+ Countries** - Curated apps and websites for destinations worldwide
- **Live Clocks** - Current time for each destination
- **Travel Checklist** - Keep track of what to pack with categories (Documents, Money, Electronics, Clothing, Health, Misc)
- **Dark/Light Theme** - Toggle between themes
- **Search & Filter** - Find apps by country or region
- **Offline Ready** - Works without internet

## Usage

Open `index.html` in any web browser.
